Task 1: Explain in your own words the following questions, write it and share answers with your mentor.

1. What is a variable? Write examples from real life.
Змінна — іменована область пам’яті, куди можна зберігати різні значення, і це значення може змінюватися під час виконання програми. Вона має тип, назву та значення.
У повсякденному житті змінна — це будь-який характеристика, що може змінюватися.
Наприклад:
- Вік людини (може змінюватися з часом)
- Температура на вулиці (міняється протягом дня)
- Швидкість автомобіля (залежить від руху)




2. What are the main types of loops, and when you think its helpful to use it?
     for - використовується, коли відома кількість ітерацій або коли потрібно пройтися по масиву чи іншій колекції з індексом.
Приклад:
for (let i = 0; i < 5; i++) {
  console.log(i);
}
while - виконує цикл, поки виконується певна умова. Використовується, коли кількість ітерацій невідома наперед.

Приклад:
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

do...while - тіло циклу виконується принаймні один раз, а потім перевіряється умова. Якщо умова істинна, тіло циклу виконується знову. Якщо ні — цикл завершується.

Приклад:
let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);



3. Explain the difference between if and switch statements.

if - виконує блок коду, якщо умова істинна. Можна використовувати кілька умов через else if і в кінці else.
Підходить для перевірки діапазонів, складної логіки або коли умови різні за типом.

switch - використовується для порівняння одного значення з багатьма варіантами, коли потрібно перевірити одну змінну на точні значення.
Кожен case відповідає певному значенню. Якщо жоден case не підходить, виконується блок default. Найкраще підходить, коли потрібно обробити багато конкретних випадків.

if використовується, коли потрібно перевірити складні або різні умови(діапазони значень або логічні вирази)
if дозволяє використовувати кілька перевірок через else if
switch краще підходить, коли потрібно порівняти одну змінну з кількома конкретними значеннями. Має блок default для випадків, коли жоден варіант не підходить.

4. What is a function, write examples from real life.

Функція — це блок коду, який виконує певне завдання або обчислення і може повертати результат.
Вона приймає вхідні дані (аргументи), обробляє їх і повертає вихідні дані (результат).
Приклад
Кавоварка
Вхід: зерна кави, вода
Функція: приготувати каву
Вихід: готова кава

Калькулятор
Вхід: числа та операції (додавання, віднімання)
Функція: виконати математичну операцію
Вихід: результат обчислення

What is object, write examples from real life.
Об’єкт — це структура, яка зберігає дані у вигляді властивостей і функцій, об’єкт описує щось конкретне, що має свої властивості і може виконувати певні дії.
Автомобіль
Властивості: колір, марка, рік випуску, пробіг
Методи: завести двигун, зупинити автомобіль, прискоритись

Телефон
Властивості: модель, виробник, колір, обсяг пам’яті
Методи: зробити дзвінок, надіслати повідомлення, зробити фото

What is an array? What you can do with objects in array? Write examples from real life.

Масив — це впорядкований список елементів, які зберігаються під одним ім’ям.
Кожен елемент у масиві має свій індекс, починаючи з 0.
Масив дозволяє зберігати багато значень одночасно звертатися до них.

З обьектами масива можна:
- Зберігати багато об’єктів одного типу в одному масиві.
- Перебирати об’єкти, щоб читати або змінювати їхні властивості.
- Фільтрувати масив, щоб вибрати об’єкти за певною умовою.
- Додавати або видаляти об’єкти з масиву.
- Знаходити конкретний об’єкт за властивістю.

Приклад
Каталог товарів в інтернет-магазині
Кожен товар — об’єкт з назвою, ціною, кількістю.
Масив зберігає всі товари, які продаються.

Масив книжна полиця
Кожна книга — це об’єкт з властивостями: назва, автор, рік видання.
Масив зберігає багато книг.
